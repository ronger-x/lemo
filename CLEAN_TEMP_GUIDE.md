# Lemo Clean-Temp 功能说明

## 🧹 功能概述

`clean-temp` 命令用于清理 Windows 系统中的各类临时文件，释放磁盘空间。

## 📝 使用方法

### 基础清理（仅系统临时文件）
```powershell
lemo clean-temp
```
清理内容：
- `C:\Windows\Temp` - Windows 系统临时目录

### 完整清理（系统+用户+浏览器）
```powershell
lemo clean-temp --include-user
# 或简写
lemo clean-temp -i
```
清理内容：
- `C:\Windows\Temp` - Windows 系统临时目录
- `%TEMP%` - 当前用户临时文件夹
- Chrome 浏览器缓存
- Microsoft Edge 浏览器缓存
- 回收站大小统计（仅显示，不清理）

## 📊 输出信息

程序会显示以下详细信息：

### 1. 实时清理日志
```
📁 清理 Windows 临时目录: C:\Windows\Temp
   ✅ 删除: tmpfile1.tmp
   ✅ 删除: tmpfile2.tmp
   ✅ 删除: old_folder
   ⚠️  跳过: locked_file.tmp (文件被占用或权限不足)
   ... 还有 150 个文件/文件夹已删除
```

### 2. 分区统计
每个清理的目录都会显示：
- 删除的文件/文件夹数量
- 跳过的文件/文件夹数量
- 释放的空间大小（MB）

### 3. 总计统计
```
📊 清理统计:
   总删除: 234 个文件/文件夹
   总跳过: 12 个文件/文件夹
   释放空间: 1234.56 MB (1.21 GB)
```

## 🔒 权限要求

- **需要管理员权限**
- 程序会自动请求 UAC 提权
- 如果拒绝提权，程序将无法运行

## ⚠️ 注意事项

1. **文件占用**: 正在使用的文件会被跳过，不会强制删除
2. **安全性**: 只删除临时文件，不会影响系统正常运行
3. **浏览器**: 清理浏览器缓存前请确保浏览器已关闭
4. **回收站**: 回收站仅显示大小，不会自动清空

## 💡 使用建议

- 定期运行可以释放大量磁盘空间
- 首次运行建议使用基础清理，熟悉后再使用完整清理
- 清理前可以关闭所有浏览器以获得更好的清理效果
- 如果某些文件被跳过，可以重启电脑后再次运行

## 🎯 典型使用场景

1. **磁盘空间不足**: 快速释放临时文件占用的空间
2. **系统维护**: 定期清理提升系统性能
3. **浏览器慢**: 清理浏览器缓存提升浏览速度
4. **软件安装**: 安装大型软件前清理空间

## 📈 预期效果

根据系统使用情况，通常可以释放：
- 系统临时文件: 100MB - 2GB
- 用户临时文件: 200MB - 5GB
- 浏览器缓存: 500MB - 10GB

总计可能释放: **1GB - 15GB** 的磁盘空间
